# Python Calculator (CLI)

A simple command-line calculator built using Python. It performs basic arithmetic operations and keeps track of previous calculations.

## Features

* Supports basic operations: `+`, `-`, `*`, `/`, `%`
* Handles division and modulo by zero
* Keeps a history of calculations
* Option to clear history anytime
* Runs continuously until you exit

## How to Use

1. Run the script:

   ```bash
   python calculator.py
   ```

2. Enter expressions in this format:

   
   5 + 3
   10 / 2
   7 * 4
   

3. Available commands:

   * `history` → shows all previous calculations
   * `clear` → clears the history
   * `exit` → closes the program


## Notes

* Only simple expressions with two numbers are supported
* Input should follow the format: number operator number
* Works with both integers and decimal numbers

## Why I Made This

I wanted to practice basic Python concepts like functions, loops, and user input handling. Also tried to make it a bit interactive by adding history and commands.
